# 🚀 Exploração Espacial

Projeto acadêmico desenvolvido com HTML, CSS e JavaScript com o objetivo de apresentar conteúdos sobre exploração espacial de forma visual, organizada e interativa.

## 📖 Sobre o Projeto

O site reúne informações sobre missões espaciais, observatórios e curiosidades relacionadas à exploração do espaço. Além disso, possui uma funcionalidade interativa que permite ao usuário calcular seu peso em diferentes planetas e corpos celestes.

O projeto foi desenvolvido com foco em:

* Responsividade
* Organização visual
* Interatividade
* Acessibilidade
* Experiência do usuário

---

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## ✨ Funcionalidades

* Página responsiva
* Menu mobile interativo
* Galeria de imagens
* Reprodução de vídeo
* Calculadora de peso em outros planetas
* Navegação entre seções
* Layout moderno com tema espacial

## 📷 Seções do Site

## 🎯 Objetivo

Tornar o aprendizado sobre exploração espacial mais acessível, visual e interativo para estudantes e interessados em ciência e tecnologia.
